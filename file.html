檔案處理

java的輸入輸出串流：

  • 輸出

    BufferedWriter output = new BufferedWriter(new OutputStreamWriter(System.out));

    方法型態 方法 說明
    void write(String) 寫入一個字串
    void flush() 清除串流內容
    void close() 關閉串流

  • 輸入

    BufferedReader input = new BufferedReader(new InputStreamReader(System.in));

    方法型態 方法 說明
    String readLine() 讀取一行文字內容的字串
    void close() 關閉串流
