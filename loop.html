.

迴圈式

迴圈在程式設計中扮演著極重要的角色，以下將會介紹各種迴圈以及其使用方式。

  • for

    先以一個範例說明：

    程式輸出
    public class example{
            public static void main(String[] args){
                    for(int i=1;i<=5;i++){
                            System.out.println(i);
                    }
            }
    }
    		 1
    2
    3
    4
    5


    由上述範例可知，for迴圈內需要一個變數來運作，且內部的三個部位分別是：

    for(變數初始值;迴圈繼續的條件;每次結束後變數的變化)

    也就是說只要迴圈繼續的條件為true，則會執行大括號內的程式，反之則離開迴圈。
    每當迴圈執行了一次便會執行第三區域的程式（每次結束後變數的變化），並再次判斷是否達成第二區域的條件。

    補充說明：
    • 先宣告變數與在小括號內宣告變數的差別：
      範例：

      先宣告變數 在小括號內宣告變數
      int i;
      for(i=1;i<=5;i++){
              System.out.println(i);
      }
      for(i=1;i<=5;i++){
              System.out.println(i);
      }
      		 for(int i=1;i<=5;i++){
              System.out.println(i);
      }
      for(int i=1;i<=5;i++){
              System.out.println(i);
      }


      由於若在括號內宣告變數的話，每一次使用迴圈都需要重新宣告一次，因此可以先在外面宣告以便使用。
      當然習慣是因人而異。

  • while, do-while

    先以兩個範例來說明：

    • 範例一

      分類 程式 輸出
      while public class example{
              public static void main(String[] args){
                      int x = 1;
                      while(x<=5){
                              System.out.println(x);
                              x++;
                      }
              }
      }
      		 1
      2
      3
      4
      5
      do-while public class example{
              public static void main(String[] args){
                      int x = 1;
                      do{
                              System.out.println(x);
                              x++;
                      }
                      while(x<=5);
              }
      }
      		 1
      2
      3
      4
      5

    • 範例二

      分類 程式 輸出
      while public class example{
              public static void main(String[] args){
                      int x = 6;
                      while(x<=5){
                              System.out.println(x);
                              x++;
                      }
              }
      }
      do-while public class example{
              public static void main(String[] args){
                      int x = 6;
                      do{
                              System.out.println(x);
                              x++;
                      }
                      while(x<=5);
              }
      }
      		 6

    由上述範例可知，只要while後面的小括號條件為true，則程式將會一直執行。

    至於while與do-while的差別為do-while一定會執行一次。

  • 使用時機

    for迴圈多用在已知且低次數的迴圈。
    while迴圈多用在未知或高次數的迴圈。

  • 無窮迴圈

    要產生無窮迴圈可以使用while(true)的方式達成，但需要注意的是，
    若程式陷入無窮迴圈將會停止運作，因此需搭配break;來離開迴圈。

  • break, continue

    break：當迴圈中遇到break時將會直接離開迴圈。
    continue：當迴圈中遇到continue時將會略過接下來的程式，直接進行下一次迴圈。
