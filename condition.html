條件式

在程式中，達成某條件才執行以下程式的用途非常廣，以下將會介紹if及else的用法。

  • if, else, else if

    範例：

    範例程式 第一輸入（x值） 第二輸入（y值） 輸出
    import java.util.Scanner;

    public class example{
            public static void main(String[] args){
                    Scanner sc = new Scanner(System.in);
                    int x = sc.nextInt();
                    int y = sc.nextInt();
                    if(x<y){
                            System.out.print("x比y小");
                    }
                    else if(x>y){
                            System.out.print("x比y大");
                    }
                    else{
                            System.out.print("x等於y");
                    }
                    sc.close();
            }
    }
    		 35x比y小
    53x比y大
    44x等於y


    如以上範例，只要if後面的括號達成條件（true），就會執行後面大括號中的程式。
    反之若不達成條件（false），就不會執行後面大括號中的程式，且若其後還有else if的條件，就繼續判斷下一個條件是否達成。
    若if及else if的條件皆不成立，則執行else內的指令（如果有else的話）。

    備註：若執行的程式僅有一行，可省略大括號：

    if(a>b){
            System.out.print("a>b");
    }

    		 等同         if(a>b)
            System.out.print("a>b");

  • switch

    若需要對單一變數做多個比對時，可以使用switch，以下為範例：

    範例程式 輸入（x值） 輸出
    import java.util.Scanner;

    public class example{
            public static void main(String[] args){
                    Scanner sc = new Scanner(System.in);
                    int x = sc.nextInt();
                    switch(x){
                            case 0:
                                    System.out.print("x is 0");
                            break;
                            case 1:
                                    System.out.print("x is 1");
                            break;
                            case 2:
                                    System.out.print("x is 2");
                            break;
                            case 3:
                                    System.out.print("x is 3");
                            break;
                    }
                    sc.close();
            }
    }
    		 0 x is 0
    1 x is 1
    2 x is 2
    3 x is 3


    如以上範例，switch後面的括號填入欲判斷的變數，
    接著在大括號內以case和break依需求填入即可。
