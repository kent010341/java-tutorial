常用變數方法

在此章節將會介紹變數常用的方法。

  • 隨機（random）

    在java中要使用隨機亂數產生器需呼叫一個方法：

    Math.random();

    需要注意的是，java中的隨機數是一個介於0到1的double型態浮點數。

    因此，若需要一個0~10間的隨機數，我們可以輸入像這樣的指令：

    int number = (int)(Math.random()*1000) % 10;

    若需要一個介於整數a到整數b間的隨機數（假設1000>a>b），我們可以輸入像這樣的指令：

    int number = ((int)(Math.random()*1000) % (a-b)) + b;

  • 次方（power）

    在java中要使用次方項需要使用一個方法：

    Math.pow(double, double);

    該方法內填入的兩個值皆須是double型態，輸出的值也是double型態，填入的值前者為底數項，後者為指數項。

    以下範例將會解釋型態的衝突與解決方法：

    int x = 3;
    System.out.print(Math.pow(x, 3));    		 Error
    int x = 3;
    System.out.print(Math.pow((double)x, 3));    		 27

  • 轉型

    在程式設計中，常常會因為要將兩個以上的變數作互動，或是需要使用方法（Method）而需要轉型，以下將介紹各種轉型方法。

    • 變數互轉

      若變數原型態與欲改變之型態為整數與浮點數型態，在最前方加上欲改變之型態及括號即可。
      範例：

      int x = 5;
      double y = (double)x;

    • 字串與算術變數轉換

      若是需要字串與算術變數轉換則有各個方法能使用，以下表格條列之：

      算術變數 轉換方法 ⇔ 轉換方法 字串
      類型 變數型態
      整數 byte Byte.toString(byte); ⇔ Byte.parseByte(String); String
      short Short.toString(short); Short.parseShort(String);
      int Integer.toString(int); Integer.parseInt(String);
      long Long.toString(long); Long.parseLong(String);
      浮點數 float Float.toString(float); Float.parseFloat0(String);
      double Double.toString(double); Double.parseDouble(String);


      範例：
      int x = 5;
      String y = Integer.toString(x);
      int z = Double.parseDouble(y);
